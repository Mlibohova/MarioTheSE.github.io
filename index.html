Frog Jumper Game

Pro Frog Jumper is a simple 2D arcade game that simulates a Frog who cannot swim that jumps from lily pad to lily pad in order to get points. The more difficult the jump, the more rewarding it is for the player, so they have to balance risk versus reward in order to get a high score on the personal leaderboards. The game will feature a basic user interface for getting in and out of the game quickly, as well as the ability to view the leaderboards. Additional features are planned such as different game modes, skins, different objectives and goals to add some variety to the gameplay, and other features to help make the game more fun and enjoyable for the user. This game was made on a private repository with a team and cannot be shared or linked to
